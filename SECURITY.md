# Política de Segurança

## Reportar uma Vulnerabilidade

**Não abra uma issue pública para reportar vulnerabilidades de segurança.**

Se você identificou um problema de segurança neste repositório, reporte de forma responsável:

1. Acesse a aba **Security** do repositório no GitHub.
2. Clique em **"Report a vulnerability"** (Advisories privados).
3. Descreva o problema com o máximo de detalhes possível.

Você receberá uma resposta em até **5 dias úteis**. Trabalharemos com você para entender e corrigir o problema antes de qualquer divulgação pública.

## Escopo

Este repositório contém **exclusivamente arquivos de texto** (Markdown, YAML). Não há código executável, dependências de runtime ou dados de usuário armazenados.

Ainda assim, vulnerabilidades relevantes incluem:

- Instruções em skills que possam induzir um agente a executar comandos prejudiciais
- Instruções que contornem controles de segurança de agentes de IA
- Vazamento acidental de informações sensíveis em templates ou exemplos

## O que NÃO é uma vulnerabilidade neste contexto

- Erros de digitação ou formatação em Markdown
- Sugestões de melhoria em skills existentes
- Incompatibilidade com uma versão específica de ferramenta

Para esses casos, abra uma [issue](https://github.com/mpr-org/agent-skills/issues/new/choose) normalmente.
