# agent-skills

Repositório público de skills para uso em conjunto com agentes de desenvolvimento.

## O que é este repositório?

Este repositório reúne um conjunto de **skills** (habilidades) reutilizáveis, projetadas para serem utilizadas com agentes de desenvolvimento de software baseados em IA (como GitHub Copilot Coding Agent, entre outros).

Cada skill representa uma capacidade ou conjunto de instruções que pode ser carregado por um agente para executar tarefas específicas de forma padronizada e consistente.

## Estrutura

```
skills/
  <nome-da-skill>/
    SKILL.md   # Descrição e instruções da skill
```

## Como usar

Cada skill possui um arquivo `SKILL.md` com:
- Descrição do que a skill faz
- Pré-requisitos e contexto necessário
- Passos que o agente deve seguir

## Contribuindo

Sinta-se à vontade para contribuir com novas skills ou melhorar as existentes abrindo uma _pull request_.
